# What is next presentation

Next presentation is a simple but highly customizable and flexible presentation framework based on react, nextjs, javascript and mdx.
It allows you to quickly create presentations using markdown format, create your own theme and add custom components via jsx.

## In short

If you are fluent in markdown you can create a beautiful presentation within minutes.
