# Media

-   Messages may contain media i.e. attachments
-   A variety of media types are supported
-   The list of types is always adapting and changing

```
AUDIO
DOCUMENT
PHOTO
STICKER
VIDEO
ANIMATION
VOICE
VIDEO_NOTE
CONTACT
LOCATION
VENUE
POLL
WEB_PAGE
DICE
GAME
```

<Note>
    ```SELECT COALESCE(type,"RAW") as type, COUNT(*) * 100.0/ sum(COUNT(*)) over
    () as percent FROM messages GROUP BY type; ```
